# signet-ffi

C-compatible FFI bindings for the [sig-net](../sig-net/README.md) protocol library.

## Build

```bash
cargo build -p signet-ffi --release
```

Output:
- `target/release/libsignet_ffi.a` — static library
- `target/release/libsignet_ffi.dylib` — dynamic library (macOS)
- `include/signet.h` — C header (auto-generated by cbindgen)

## API

All functions return `int32_t` (`0` on success, negative on error).

```c
#include "signet.h"

uint8_t k0[32];
signet_derive_k0_from_passphrase("Ge2p$E$4*A", 10, k0);

uint8_t sender_key[32];
signet_derive_sender_key(k0, sender_key);
```

## C++ Usage

Link with the static or dynamic library and include `signet.h`:

```bash
c++ -std=c++17 -I include \
    your_code.cpp \
    target/release/libsignet_ffi.a \
    -o your_program
```

See [examples/signet-ffi](../examples/signet-ffi/) for a complete working example with Makefile.
